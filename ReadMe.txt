1. Apache Maven 3.3.9 – 
set path=<directory in which apache maven 3.3.9 is installed> 
Or 
Set environment variable “path” with value as apache maven installation directory.


2. Java 7.0 or higher version
Set path to jdk/bin directory 
Set JAVA_HOME to Java installation directory.


Commands to compile & run the application:
1.	mvn compile
2.	mvn package
3.	mvn spring-boot:run
Before running these commands, user is required to change to directory in which application is extracted and having pom.xml directly under it.
